# 📊 Business Sales Dashboard (Task 1)

## 📌 Objective
Analyze sales data to identify trends, top products, and business opportunities.

## 🛠 Tools Used
- Tableau

## 📊 Dashboard Features
- Sales Trend Over Time
- Top 10 Products by Sales
- Category-wise Performance
- Region-wise Sales Analysis
- Interactive Filters

## 📊 Key Insights
- Sales show an increasing trend, especially towards year-end
- Technology category generates the highest revenue
- A few top products contribute significantly to total sales
- West region performs the best among all regions

## 🚀 Recommendations
- Focus on high-performing categories like Technology
- Improve performance in low-sales regions
- Maintain stock for top-selling products
- Plan promotions during low-sales periods

## 📁 Files Included
- Tableau Dashboard (.twbx)
- Dataset (.csv)
